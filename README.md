## 📻 Features
- 🎵 Live streaming of 200+ Kenyan FM radio stations
- 📊 Station categorization by genre, region, and popularity
- ⭐ Favorite stations for quick access
- 🎧 Background playback support
- ⏰ Sleep timer for bedtime listening
- 🔍 Station search and discovery
- 📈 Trusted by over 100,000+ listeners across Kenya and globally

## 🏗️ Technical Highlights
- Built with Kotlin and Android SDK
- ExoPlayer for high-quality audio streaming
- Material Design UI with custom themes
- Room database for local station caching
- MVVM architecture with LiveData
- Retrofit for API integration
- Firebase Analytics for usage insights
